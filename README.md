# DOCKER

Docker Compose para inicialização dos containers de Ngix e MySQL.
Arquivo criado para a cadeira de Infraestrutura e Cloud Architecture do curso de Arquitetura de Soluções Digitais da Faculdade Impacta de Tecnologia.

## Pré-Requisitos:

    - Docker
    - Docker-compose

## Instalação

Para subir os containers basta executar o seguinte comando:

```
$ docker-compose up
```

Para manter em segundo plano basta executar com a flag "-d", como mostra a seguir:

```
$ docker-compose up -d
```
